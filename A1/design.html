Design

Icons and Fonts

External CSS is used to unify multiple pages into the same style to prevent the sense of fragmentation. To make the site looks consise, the use of icons allows web pages to be used more efficiently. The use of shadow can also imply hierarchical spatial relationships. Fonts are set to sans-serif and space between letters are adjusted to a proper width.

Layouts

There is also a top navigation bar for jumping pages easily and a side navigation bar to have an overview of the content which helps users quickly locate the part they want to read. Hierarchy is communicated through differences in font weight, size and indent.


Usability

This site has been tested in lighthouse.All 4 items are green.

Accessibilty

This website has a clear navigation and table of contents. Use CSS to make responsive web pages, the site has optimised the readability on mobiles or other small width devices. When cursor hovering over a button, button will turn gery to give hints to guide users click on it.

Performance

In my test, first contentful paint is at 0.6s. The loading time of full page is 1.4s.


Colour

Considering the there will be some pictures, which are colorful, so the site uses light colours, simple but can have a neat visual. This site has avoided using pure white and pure black to maintain a comfortable contrast, not too large or small, which is validated by lighthouse.This is also readable if having colour blindness.
