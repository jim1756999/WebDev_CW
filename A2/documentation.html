Summary

To develop this website. Vscode is used as the text editor to create this side, and the viewing of the website using Chrome. Git is used for version controlling, which will simplify if codes added is encountering problems. Some plug-ins are also used to boost the efficiency, live-server is implied to get the output in realtime, TODO Tree and formatter is also used. In this website, Roboto and Fontawesome are used to make the site looks elegent with a clear font and intuitionistic icons. Lighthouse and w3 validator are used to check the website.

Accessibility

This website has a clear navigation. CSS is used to make responsive web pages, to optimised the readability on mobiles or other small width devices. When cursor hovering over a button. As an indicator, button will turn gery when users hover on it, to suggest them it can be clicked. Considering the there will be some pictures, which are colorful, so the site uses light colours, simple but can have a neat visual. This site has avoided using pure white and pure black to maintain a comfortable contrast, not too large or small. This is also readable if having colour blindness. The accessibility is validated by Lighthouse.

Security

Security issues are mainly considered in server configuration and back-end. The main security threats are SQL injection and cross-site scripting attacks. In order to protect security, in this website, the data will be checked before submitting the form to prevent the data from being tampered with by externally injected javascript. In addition, divide the content into a clear and reasonable level system, such as the division of columns and their relationships, the level of web pages and their relationships, and the path setting of links. For instance, all links in the webpage use relative paths to avoid cross-site attacks.

Enhancement

Make the content of the website richer and add more materials, pictures and videos. Use more variety of layouts in the website. Perform more accurate calculations on color contrast and layout. The web page looks more concise and clear.
